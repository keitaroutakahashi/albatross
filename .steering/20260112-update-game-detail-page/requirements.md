# 要求内容

## 概要

試合詳細ページ（/games/[id]）を、プロ野球の試合結果ページ（ヤクルトスワローズ公式サイト）を参考に、より詳細な試合情報を表示できるよう更新する。

## 背景

現在の試合詳細ページは基本的なスコアボードのみを表示している。草野球チームの試合結果を記録・閲覧する際に、戦評、責任投手、本塁打、試合統計などの詳細情報を表示することで、チームメンバーが試合内容をより深く振り返ることができるようになる。

## 実装対象の機能

### 1. 戦評（試合サマリー）セクション

- 試合の流れを要約したテキストを表示
- 見出しと本文で構成

### 2. 責任投手・本塁打セクション

- 勝利投手、敗戦投手、セーブ投手を表示
- 本塁打者を表示

### 3. 試合統計セクション

- 試合時間を表示
- 参加人数を表示

## 受け入れ条件

### 戦評セクション

- [ ] 試合の戦評（summary）が表示される
- [ ] 戦評がない場合はセクション自体が表示されない

### 責任投手・本塁打セクション

- [ ] 勝利投手、敗戦投手、セーブ投手が表示される
- [ ] 本塁打者が表示される
- [ ] 該当データがない項目は「-」または非表示

### 試合統計セクション

- [ ] 試合時間が表示される
- [ ] 参加人数が表示される

### 全体

- [ ] 既存のスコアボード表示が維持される
- [ ] レスポンシブデザインが適用される
- [ ] 既存のカラーパレット・デザインシステムに準拠

## 成功指標

- 試合詳細ページでプロ野球公式サイトに近い情報量を表示できる
- チームメンバーが試合の詳細を把握しやすくなる

## スコープ外

以下はこのフェーズでは実装しません:

- 打撃成績表（選手ごとの打席成績）
- 投手成績表（投手ごとの詳細成績）
- スターティングメンバー表示
- データベース連携（引き続きダミーデータを使用）
